# ☕ Highlands Coffee Management System

## 1. Giới thiệu dự án

Highlands Coffee Management System là hệ thống hỗ trợ quản lý quán cà phê được xây dựng nhằm tối ưu hóa quy trình vận hành và nâng cao trải nghiệm người dùng. Hệ thống hỗ trợ các chức năng như quản lý sản phẩm, đơn hàng, khách hàng, thanh toán và quản trị hệ thống.

Dự án được phát triển với mục tiêu mô phỏng quy trình quản lý thực tế tại chuỗi cà phê Highlands Coffee, đồng thời áp dụng các công nghệ hiện đại trong phát triển phần mềm.

* Theo dõi trạng thái đơn hàng

* Quản lý danh mục

* Quản lý nhân viên
